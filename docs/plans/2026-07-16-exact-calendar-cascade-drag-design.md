# 정확한 시간 캘린더와 연쇄 이동 설계

## 목표

1~3단계에서 자동 생성된 시간표는 초안으로 사용한다. 4단계에서는 사용자가 활동 블록을 원하는 요일과 정확한 시각으로 옮겨 최종 시간표를 완성한다. 드래그 경험은 가로로 요일이 바뀌고 세로로 시간이 바뀌는 주간 캘린더 방식으로 제공한다.

## 시간 배치

- 주간 캘린더는 가장 이른 기상 시각부터 가장 늦은 취침 시각까지의 세로 시간축을 표시한다.
- 시간축은 1시간 눈금과 30분 보조선을 보여주며, 실제 배치는 10분 단위로 맞춘다.
- 블록을 드래그하면 기존 길이를 유지한 채 포인터가 가리키는 요일과 시작 시각을 실시간으로 표시한다.
- 드롭 위치가 기상·취침 범위를 벗어나면 저장하지 않고 원래 자리로 돌려보낸다.

## 연쇄 이동

- 새 위치와 겹치는 이동 가능한 블록은 같은 요일의 뒤 시간으로 순서대로 밀린다.
- 연쇄 이동 중에도 각 블록의 길이와 기존 순서를 유지한다.
- 고정 일정과 잠긴 일정은 이동하지 않는다. 새 블록이 직접 겹치면 드롭을 거부한다.
- 뒤로 밀리는 블록이 고정·잠금 일정에 닿으면 그 일정이 끝난 다음 빈 시간부터 이어서 배치한다.
- 연쇄 이동 결과가 취침 시간을 넘으면 전체 드롭을 취소한다.
- 다른 요일로 자동 이동시키지 않는다.

## 피드백과 되돌리기

- 드래그 중 목표 블록은 정확한 시각 위치에 표시하고, 뒤로 밀릴 블록은 반투명 미리보기로 이동 결과를 보여준다.
- 불가능한 위치는 붉은색으로 표시하고 이유를 안내한다.
- 드롭 애니메이션과 밀리는 블록의 위치 전환에는 부드러운 easing을 적용한다.
- 완료 후 `N개 일정이 뒤로 이동했어요 · 되돌리기`를 제공하고 한 번의 클릭으로 드롭 직전 결과를 복원한다.

## 검증

- 정확한 요일·시각 배치, 단일 충돌, 다중 연쇄 충돌, 고정·잠금 우회, 취침 시각 초과를 단위 테스트한다.
- 데스크톱 포인터와 모바일 길게 누르기에서 목표 시각 미리보기 및 드롭 결과를 확인한다.
- 수정된 `ScheduleResult`가 방학 전체 보기와 PNG/PDF/PPTX 출력에 그대로 반영되는지 확인한다.

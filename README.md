# 방학한칸

초등학생과 보호자가 함께 여름방학 생활계획표를 만드는 모바일 우선 웹 서비스입니다. 회원가입이나 외부 API 없이 브라우저에서 바로 작동하며, 입력 내용과 생성 결과는 `localStorage`에 저장됩니다.

## 주요 기능

- 4단계에서 활동 블록 전체를 원하는 요일의 오전·오후로 끌어서 이동
- 드래그 중 배치 가능한 구역과 실제 시작·종료 시각 미리보기
- 고정 일정과 잠긴 블록은 이동하지 않도록 보호
- 여행·체험학습을 시작일과 종료일로 한 번에 등록하고 1일·2일·3일 빠른 선택
- 기상·취침 시각을 10분 단위로 설정
- 새 계획은 활동 카드가 모두 선택되지 않은 상태로 시작
- `꼭 넣어줘!` 활동은 요청 횟수를 전부 배치하며, 불가능하면 누락된 결과를 만들지 않고 안내
- 마지막 단계의 요일별 `+` 버튼으로 부모 일정 카드를 직접 추가
- 반복 주간 일정과 방학기간 전체 일정 보기
- 특별한 날은 해당 날짜의 반복 일정을 대체
- 현재 주간 일정 PNG 저장 및 인쇄
- 주간 또는 방학 전체 일정을 PDF로 저장
- 텍스트와 도형을 PowerPoint 안에서 직접 수정할 수 있는 PPTX 저장

## 실행

```powershell
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 검증

```powershell
npm test
npm run typecheck
npm run build
```

PDF는 가로형 A4 한 페이지에 한 주씩 생성됩니다. PPTX는 시간표를 이미지로 붙이지 않고 PowerPoint의 텍스트 상자와 도형으로 구성합니다.
